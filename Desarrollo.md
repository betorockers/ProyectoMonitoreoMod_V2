## 🌌 CONTEXTO Y ESTÉTICA DEL PROYECTO
Estamos construyendo una aplicación híbrida de monitoreo táctico con una interfaz **full moderna, distópica y ciberpunk**. El diseño debe ser tan vanguardista y robusto que debe verse fresco, vivo y futurista incluso dentro de 10 años.
- **Paleta de Colores Base:** Fondo negro absoluto (`#000000`), textos en blanco puro (`#FFFFFF`).
- **Acentos Visuales:** Cian eléctrico y verde menta brillante aplicados estrictamente en animaciones, métricas dinámicas, flujos de datos y visuales interactivos de alto impacto.
- **Inspiración:** Terminales de operaciones tácticas, interfaces de control de redes oscuras y HUDs militares distópicos.

## 🛠️ ARQUITECTURA TÉCNICA DEL SISTEMA
La aplicación monitorizará en tiempo real: redes, servidores, sitios web, cámaras IP y streams de video. Debe cumplir con los siguientes pilares de ingeniería:

1. **Backend (Python 3.11+ / FastAPI):**
   - Arquitectura limpia (Clean Architecture), desacoplada y orientada a objetos (POO).
   - Implementación obligatoria de **Multithreading** y **AsyncIO** para la ejecución concurrente de pings, peticiones HTTP, análisis de sockets y procesamiento de streams de video sin generar retrasos ni bloqueos en el sistema.
2. **Frontend (React / Next.js con App Router):**
   - Interfaz ultramoderna, optimizada para baja latencia. Usar estados de carga fluidos y animaciones optimizadas para GPU (Cian/Verde menta).
   - **Integración Cartográfica:** Integrar y adaptar visualmente los mapas vectoriales interactivos del repositorio `https://github.com/AnmolSaini16/mapcn` para geolocalizar servidores, nodos de red y cámaras IP dentro de la interfaz distópica.
3. **Compilación y Empaquetado Nativo Profesional:**
   - **Nuitka:** Compilación total del ecosistema Python a ejecutables binarios nativos de alto rendimiento, protegiendo el código fuente y optimizando el consumo de recursos.
   - **Inno Setup (`installer.iss`):** Creación de un instalador para Windows de nivel empresarial, configurado a tope.
   - **Autofirmado (Anti-SmartScreen):** Incluir la automatización de la generación de certificados intermedios y scripts de firma digital (`SignTool`) para todos los binarios y el instalador, evitando alertas de "Editor desconocido" o bloqueos de Windows Defender.

## ⚙️ REGLAS DE OPERACIÓN (STRICT)
- Priorizar la calidad arquitectónica, el patrón SOLID y la escalabilidad sobre la rapidez.
- El diseño visual no es negociable: debe evocar una terminal ciberpunk viva de alta tecnología.
